Prework Study Guide

HTML

  • Head element contains information about the webpage.
  • Body element represents the visible content shown to the user
  • h1 through to h6 elements represent the size of a heading. Header 1 being the biggest
  • p element represents a paragraph or block of text. Used frequently to wrap most texts on a webpage
  • ul unordered list
  • ol ordered list
  • li list items
  • img contains iformation about images on the webpage
  • src attribute defines the location of the image.
  • a element (anchor) creates links to the same, or other webpages.
  • href attribute points to the URL for the link; example: More About Giant Pandas

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • pwd - displays the current working directory
  • ls - displays the list of files and folders
  • cd - navigates between directories in the path
  • cd .. navigates backwards
  • cd ~ - navigates to user root directory
  • mkdir - creates a new directory (creates a new folder)
  • touch - creates a file inside a directory (eg touch insert_name.txt)
  • git add -A
  • git commit -m "insert message here"
  • git push
  • git clone
  • git reset --hard -nukes everything, always make sure everything you want has been committed before nuking
  • git checkout -b insert branch name creates a new branch and switches to it
